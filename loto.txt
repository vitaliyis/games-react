	- количество игроков: человек и компьютер
	- количество карточек у игрока 2
	- скорость игры в секундах: 1 - 5

1. Разработка карточки
2. Экран делится на две части

На вход CardsList будет приходить количество карточек, массив с рандомными значениями [1-27]
Массив из 27 ячеек должен иметь 15 заполненых значений в диаплзоне 1-99
Состояние лото:
	state = {
		quantityCards: null,
		valuesCardsPeople: null,
		valuesCardsComputer: null,
		numberFilledCardPeople: null,
		numberFilledCardComputer: null
		}

Заполнение: 6 по 2 и 3 по 1
Рандомность делаю в два шага:
	1. Выбираю 3 по 1 или 6 по 2 девять итераций
	2. ДЛя каждого диапозона уже известно количество раз рэндомов и выбираю конкретно число
	
[0, 2, 0, 19, 0, 0, 0, 21, 0, 0, 31, 32, 0, 48, 44, 0, 57, 50, 0, 69, 61, 0, 74, 73, 81, 0, 87]
[1,10, 19,2,  11,20,3,12,21]
		
		
Алгоритм игры:
1. Установка setInterval. Время и случайное число.
2. Когда работает setInterval данные передаются на табло
3. После каждого тика идет проверка на победу. Если кто-то победил игра останавливается и появляется сообщение с поздравлениями
	Как проводится проверка:
		- после удаления числа из карточки проверяем все ли у карточек нули или нет
4. Доложна быть кнопка сброса игры
